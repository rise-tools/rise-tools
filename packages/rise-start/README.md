

### Commands

#### `rise-start dev` - Start the development server

```
Options:
  -ws                Websocket server
  -p, --port [port]  Port for dev server (default: 3500)
  -m, --host [host]  dev server host, can be localhost, lan, tunnel (default: "lan")
  -h, --help         display help for command
```
